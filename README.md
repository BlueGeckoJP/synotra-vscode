# synotra-vscode README

synotra-vscode adds the Synotra language support to VSCode

[Synotra language](https://github.com/Kakeru3/synotra)

## How to use

You need bundle this extension because it is under development currently

```bash
# Install vsce
npm install -g vsce

# Bundle
vsce package

# Then install .vsix
```

## Features

- Syntax highlighting
- Auto completion
